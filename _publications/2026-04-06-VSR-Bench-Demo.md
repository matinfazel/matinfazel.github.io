We introduce a live demo of VSR-Bench, an interactive, browser-native platform for end-to-end evaluation of real-time receiver-side VSR within WebRTC. The demo integrates state-of-the-art VSR models using TensorFlow.js and supports multiple execution architectures and GPU backends (WebGL/WebGPU). Users can dynamically configure codecs, models, and runtime settings while observing real-time feedback on latency, perceptual quality (VMAF), and network conditions. This interactive setup reveals critical trade-offs between computation, latency, and QoE, offering actionable insights into deploying VSR in practical RTC environments.
